# Ancient-Chinese-Architectural-Achievements
# Part1. 作品设计

## 作品主题

### 主题陈述
- **主题**：中国古代建筑成就——《园林小百科》
- **类型**：一款以苏州古典园林为主题、融合科普知识的轻量级文化科普小游戏

### 作品简介
作品以拙政园为叙事舞台，通过生动有趣的交互与探索，系统性地呈现中国古代建筑的艺术成就。玩家将在游戏中扮演一名意外穿越至拙政园的现代访客，在一位"穿越系统"的引导下，逐步了解园林中的建筑布局以及空间意境营造等多方面的知识。

### 灵感来源
作品的灵感来源于近年来广受欢迎的"穿越"题材影视与文学作品。我们设想玩家因一次偶然的机缘，穿越至拙政园，成为其建造过程中的一名见证者与参与者。在游戏中，玩家将跟随一位"系统"，解开园林中隐藏的建筑谜题，在过程中逐步领悟中国古代建筑中蕴含的天人合一思想、工匠精神与诗意栖居理想。

### 意义
能够让人们想起对中华建筑遗产的关注与热爱，了解中国古代建筑成就，体会中国古代劳动人民的智慧，增强文化自信。
